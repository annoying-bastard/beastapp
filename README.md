# BEASTAPP

### Introduction

Beastapp is a build-ready electron app for testing purposes.

### Usage

```
git clone https://github.com/annoying-bastard/beastapp
cd beastapp
npm i
```

### Build

To build a executable packaged app for your os

`npm run build`

in the project root directory
